Window下Docker环境部署(三)

命名卷部署redis7.0方案

1 创建命名数据卷

特点

  • 数据存储在Docker托管路径（通过\wsl$\docker-desktop-data...访问）
  • 无需手动配置Windows目录权限，适合快速部署

2.启动SQL Server容器

关键参数说明：

  • -v redis_vol:/data：将容器内的/data映射到命名卷
  • 容器停止/重启不会删除数据，但执行docker rm会销毁未挂载的容器层数据

3.验证部署状态

开发环境
docker Window redis
